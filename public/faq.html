How can I load my hist performance line chart?

Every time you re-enter or reload the web page, you should always click the green "Load Chart Data" before hitting the blue "AuM" button, to ensure the browser can load the data for display.


How can I change the number of days displayed?

The red rectangle represents the range option, which you can modify to any number you want.



In The HistPrices tab, how do I view the chart data?

Basically there are two ways:

  • 1. Searching by Security ID
    • 1. Enter your search security ID into the search input text box
    • 2. Then click the result you want to display.
  • 2. Display multiple securities at the same time through the multi-select bar.
    • 1. Select the securities you want to see them together.
    • 2. Then click the blue "2. Show Charts" button sitting on the top, now you can see the result.

In both ways, before you do anything, load the data by clicking the green "Load Charts Data" button first!
